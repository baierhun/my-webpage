Styles

Style refers to the visual presentation of elements on a webpage, controlled by Cascading Style Sheets (CSS). CSS allows developers to define how HTML elements should appear, including properties like color, font size, background, layout, and more.

  • style="..."
    • add the style attribute to any tag
  • propert-name: value;
    • add properties to the style attribute in this format
  • Some example properties:
    • color: darkblue;
    • background-color: aliceblue;
    • text-align: center;
    • width: 100%;
    • padding: 0% 10%;
    • font-size: larger;
    • border: 1px; border-style: solid;
