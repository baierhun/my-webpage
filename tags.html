Tags

A tag is a piece of markup language used to define and structure elements within an HTML document. Tags are essentially instructions to a web browser, telling it how to interpret and display content on a web page.

  • <html style="background-color:midnightblue; padding: 0% 20%;"> - needed at the top
  • <head> - put the title (and other page info) in here
  • <title> - the words in the browser tab
  • <body style="background-color: aliceblue;">
  • <h1 style="color: cornflowerblue; text-align: center;"> - h1 (biggest) through h6
  • <p style="color: darkblue; padding: 0% 10%;"">
  • <br> - line break for adding a horizontal space

  • <hr> - horizontal rule creates a horizontal line
  • <a href="anotherPage.html"> - this displays a link
  • <img src="totoro.png" width="50px"> - this displays an image
